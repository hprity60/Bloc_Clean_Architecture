## For developers
> To clone locally
```
$ git clone git@github.com:luciano01/favorites_movies.git
$ cd favorites_movies
```
> Open the project in an editor, if you use VS Code, follow this
```
$ cd favorites_movies
$ code .
```

> To run, follow this on terminal
```
$ flutter clean
$ flutter pub get
$ flutter run --release
```
